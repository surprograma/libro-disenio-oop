Guía de instalación IntelliJ Idea

Prerrequisito: Java Development Kit (JDK)#

Para poder ejecutar el IntelliJ Idea y los programas que escribas en Kotlin, vas a necesitar instalar el Java Development Kit (JDK). En base a pruebas que hemos realizado en distintas computadoras, recomendamos enfáticamente instalar la versión 11: ni una más reciente ni una más antigua.

Comprobar si el JDK ya está instalado#

Puede que ya lo tengas de otro proyecto, o incluso que haya venido instalado en tu sistema operativo. Podés comprobarlo de la siguiente manera:

  • Ubuntu/Debian
  • Windows

Abrí una terminal y ejecutá el comando java -version.

Si no tenés ninguna versión instalada, la salida se verá así:

$ java -versionNo se ha encontrado la orden «java», pero se puede instalar con... (varias opciones)

Caso contrario, la respuesta será similar a la siguiente:

$ java -versionopenjdk version "11.0.11" 2021-04-20OpenJDK Runtime Environment (build 11.0.11+9-Ubuntu-0ubuntu2.20.04)OpenJDK 64-Bit Server VM (build 11.0.11+9-Ubuntu-0ubuntu2.20.04, mixed mode, sharing)

Vamos a ejecutar un comando en la consola. Para abrirla, presioná a la vez las teclas Windows + R, escribí cmd en la ventana que aparece y luego presioná Enter.

Dentro del cmd escribí java -version y presioná Enter.

En el caso de no tener ninguna versión instalada, la salida se verá así:

C:\Users\Pepita>java -version"java" no se reconoce como un comando interno o externo, programa o archivo por lotes ejecutable

Caso contrario, será similar a la siguiente:

C:\Users\Pepita>java -versionjava version "11.0.11" 2021-04-20 LTSJava(TM) SE Runtime Environment 18.9 (build 11.0.11+9-LTS-194)Java HotSpot(TM) 64-Bit Server VM 18.9 (build 11.0.11+9-LTS-194, mixed mode)

En el caso de que no tengas ninguna versión, o tengas una que no es la 11, podés proceder a instalar el JDK 11.

Instalar el JDK 11#

  • Ubuntu/Debian
  • Windows

Ejecutá el siguiente comando:

sudo apt install openjdk-11-jre-headless

Visitá el sitio de Oracle, registrate, descargá el JDK, instalalo y reiniciá la máquina. Hasta donde sabemos, no hay una forma más sencilla de instalarlo, ni es posible usar la OpenJDK. 🤷

Verificar que se haya instalado la versión correcta#

Volvé a ejecutar los pasos descriptos en Comprobar si el JDK ya está instalado y verificá si la versión que aparece es la 11.

Si es así, ya podés Instalar IntelliJ Idea. Caso contrario, deberás indicarle a tu sistema operativo qué versión usar.

  • Ubuntu/Debian
  • Windows

Dentro de una terminal ejecutá lo siguiente:

sudo update-alternatives --config java

Lo cual debería arrojar un mensaje como el siguiente...

Existen X opciones para la alternativa de java (que provee /usr/bin/java)...

...mostrando una tabla con un número de seleccion a la izquierda de cada versión y la siguiente leyenda al final:

Pulse <Intro> para mantener el valor por omisión [*] o pulse un número de selección:

En este caso, elegí el número de opcion que correspondá a java-11-openjdk y presioná Enter.

Seguí la siguiente guía.

Instalar IntelliJ IDEA#

  • Ubuntu/Debian
  • Windows

Dentro de una terminal ejecutá lo siguiente:

sudo snap install intellij-idea-community --classic

Descargá el instalador desde el sitio oficial y seguí las instrucciones.

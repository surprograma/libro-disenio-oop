Trabajo colaborativo en IntelliJ Idea

Aprender a trabajar en equipo es vital para nuestra profesión, pero integrar nuestro trabajo puede ser un dolor de cabeza si no utilizamos las herramientas correctamente.

A continuación, presentamos dos formas de trabajo posibles: una sincrónica y la otra asincrónica. Cada equipo de trabajo decidirá cuál utilizar y en qué momentos, aunque desde este espacio recomendamos familiarizarse con ambas y elegir la adecuada para cada ocasión.

Versión asincrónica: git pull y git push#

Trabajo en equipo y resolución de conflictos de Git desde el IntelliJ Idea

La guía definitiva (o algo así) para trabajar en equipo con Git y no morir en el intento.

Versión sincrónica: Code with me#

Code with me: trabajo colaborativo remoto en el IntelliJ Idea

Una linda funcionalidad que sumó hace poco el IntelliJ, que nos permite programar en equipo _casi_ como si estuvieramos al lado.
